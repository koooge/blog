# blog
Wannabe engineer's blog https://koooge.com/

## Setting
### Netlify
Build command: `$ hugo -t hugo-theme-geppuku`
Publish directory: `public`

## Usage
### Create a new post
```
$ hugo new content/post/yyyymmdd_<title>.md
```

### preview
```
$ hugo server --theme=hugo-theme-geppaku --buildDrafts --watch
```
